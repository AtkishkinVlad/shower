Суть проекта

Сайт, где любой студент/преподаватель/компания может предложить идеи для научных работ

  • Автор определяет роли для успешного написания статьи
  • Если кому-то понравилась идея, то он может подать заявку и стать соавтором

Какие проблемы решаем

  • Упрощаем поиск темы для научной работы
  • Улучшаем коммуникации между студентами/преподавателями разных ВУЗ-ов/институтов/кафедр

Целевая аудитория

  • ВУЗы
  • Крупные компании

Типичный представитель

  • Михаил, 27 лет
  • Аспирант в институте экономики
  • Делает свой проект для автоматизации работы с базами данных
  • Хочет найти студентов, которые смогли бы помочь с технической частью и написанием документации на проект для нескольких языков

Особенности MVP

  • Запуск в рамках УрФУ

Метрики

  • Количество активных пользователей / зарегистрированных пользователей
  • Распределение тегов между техническими и гуманитарными ролями
