**[Юнит экономика](https://uecalc.com/s2918GVMbJw)**

* Выделенный сервер Selectel -> AEL10-SSD (45_900 руб за 12 месяцев)
* Подписка на приоритетную выдачу -> 289 руб (1 месяц)
* Привлечение людей -> через соц.сети УрФУ и группы институтов + рекламные банеры = ~бесплатно
* Количество студентов в УрФУ -> 57_000
* Количество людей с подпиской при конверсии -> 0,38 × (1 / 100) × 57 000 = 216,6
* Затраты на каждого студента с премиумом -> 45_990 / 216 = 212,916666667
* Прибыль с одного студента за месяц -> 289 − 212 = 77  

**Как запустить презентацию у себя**

* Установить [Node.js](https://nodejs.org/en/) 
  
  * Через пакетные менеджеры
    
    * Windows: `winget install nodejs`
    
    * MacOS: `brew install nodejs`

* Установить [Git](https://git-scm.com/)
  
  * Через пакетные менеджеры
    
    * Windows: `winget install git`
    
    * MacOS: `brew install git`

* Выполнить команды
  
  * `git clone https://github.com/AtkishkinVlad/shower`
  
  * `npm install`

  * `cd shower`
  
  * `npm run start`
